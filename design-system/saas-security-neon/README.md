# SaaS Security Neon Dark (`saas-security-neon`)

Reusable design bucket for dark, data-dense BrandGuard workflow dashboards derived from 5 reference pages.

## Included Files
- `bucket.json` - machine-readable style contract
- `tokens.css` - scoped design tokens under `.theme-saas-security-neon`
- `agent-context.md` - implementation guardrails for coding agents
- `INTEGRATION.md` - install and integration notes

## Reference Page Mapping
- 01 `table` -> table interactions and pagination
- 02 `security-overview` -> case management lifecycle
- 03 `charts` -> case detail and decision explanation
- 04 `heatmap` -> domain scan simulator
- 05 `ui-elements` -> threat trends and system overview
